2. Allgemeines

Wir verarbeiten personenbezogene Daten unter Einhaltung des Schweizer Datenschutzgesetzes (nFADP) und – soweit anwendbar – der EU-Datenschutz-Grundverordnung (DSGVO). Diese Erklärung informiert Sie darüber, welche Daten wir wann erheben, wofür wir sie verwenden und welche Rechte Ihnen zustehen.

3. Beim Besuch der Website

Unsere Website wird via Cloudflare und GitHub Pages ausgeliefert. Beim Aufruf einer Seite werden technische Daten verarbeitet, die der Browser automatisch übermittelt:

  • IP-Adresse (gekürzt oder vollständig, abhängig vom Anbieter)
  • Datum und Uhrzeit des Zugriffs
  • aufgerufene URL / Dateipfad
  • HTTP-Statuscode und übertragene Datenmenge
  • Referrer (sofern übermittelt)
  • User-Agent (Browser, Betriebssystem)

Diese Daten werden zur Bereitstellung der Website, zur Sicherheit (Angriffsabwehr) und zur statistischen Auswertung in aggregierter Form verarbeitet. Eine Zusammenführung mit anderen Daten erfolgt nicht.

4. Kontaktformular

Wenn Sie das Kontaktformular verwenden, verarbeiten wir die von Ihnen eingegebenen Daten:

  • Name
  • E-Mail-Adresse
  • Firma (optional)
  • Inhalt der Nachricht
  • technische Metadaten (IP, Land, User-Agent, Zeitstempel)

Die Verarbeitung erfolgt zur Bearbeitung Ihrer Anfrage (vorvertragliche Massnahme / berechtigtes Interesse). Die Nachricht wird per E-Mail an unsere Postfächer und intern in einen Discord-Kanal zugestellt.

5. Auftragsverarbeiter und Drittdienste

Zur Bereitstellung der Website nutzen wir folgende Dienste:

  • Cloudflare Inc. (USA) — Hosting, CDN, E-Mail-Routing, Worker-Backend für das Formular. Server können sich auch in der EU oder den USA befinden.
  • GitHub Inc. (USA) — Auslieferung der statischen Seiten via GitHub Pages.
  • Discord Inc. (USA) — Interne Zustellung eingehender Anfragen über einen privaten Webhook.

Mit diesen Anbietern bestehen entsprechende Vereinbarungen zur Auftragsverarbeitung. Daten in die USA werden auf Basis der Standardvertragsklauseln und – soweit verfügbar – des Swiss-U.S. Data Privacy Framework übermittelt.

6. Schriften und Bilder

Die verwendeten Schriften (Instrument Serif, Manrope, JetBrains Mono) werden lokal von unserem Server ausgeliefert. Es findet keine Übertragung an Google Fonts oder andere Drittanbieter statt. Bilder liegen ebenfalls lokal vor.

7. Cookies

Wir setzen keine Cookies und verwenden kein Tracking. Es werden keine Analyse- oder Marketing-Werkzeuge geladen.

8. Speicherdauer

Anfragen über das Kontaktformular werden so lange aufbewahrt, wie es zur Bearbeitung und zur Erfüllung gesetzlicher Aufbewahrungspflichten erforderlich ist. Server-Logs werden in der Regel nach 30 Tagen automatisch gelöscht.

Sie haben jederzeit das Recht auf Auskunft über die zu Ihrer Person gespeicherten Daten, auf Berichtigung unrichtiger Daten, auf Löschung und Einschränkung der Verarbeitung sowie auf Datenübertragbarkeit. Bei berechtigten Interessen unsererseits können Sie der Verarbeitung widersprechen.

Anfragen richten Sie bitte an hallo@wolf-werler.ch. Sie haben zudem das Recht, sich bei der zuständigen Aufsichtsbehörde (in der Schweiz: EDÖB) zu beschweren.

10. Änderungen
